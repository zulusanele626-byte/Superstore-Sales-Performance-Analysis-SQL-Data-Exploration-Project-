# Superstore-Sales-Performance-Analysis-SQL-Data-Exploration-Project-

The aim of this project is to analyze retail sales data to uncover business insights related to sales performance, customer behavior, product trends, and operational efficiency using SQL.

## Project Objectives

To evaluate overall sales performance across time (years & months)
To identify top-performing regions, states, and products
To analyze customer purchasing behavior and repeat orders
To measure customer value (lifetime value)
To assess shipping efficiency and delivery performance
To break down sales by segment, category, and sub-category
